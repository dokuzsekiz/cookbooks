# Vagrant

```bash
vagrant init ubuntu/trusty64    # config dosyasını oluşturur
vagrant up                      # sistemi ayaklandırır
vagrant status                  # kurulu sistemleri gösterir
vagrant ssh                     # ssh ile bağlanmak
vagrant halt                    # kapatmak
vagrant suspend                 # uyku moduna almak
vagrant resume                  # uyku modundan geri almak
vagrant reload                  # sistemi yeniden başlatmak
```
